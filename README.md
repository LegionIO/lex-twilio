# lex-twilio

Twilio communications integration for [LegionIO](https://github.com/LegionIO/LegionIO). Send SMS, make calls, and use other Twilio services from within task chains.

## Installation

```bash
gem install lex-twilio
```

## Requirements

- Ruby >= 3.4
- [LegionIO](https://github.com/LegionIO/LegionIO) framework
- Twilio account
